# 写作技法

> 来源：novelist（6条句法红线+推拉镜头）+ openclaw（反AI写作v4+具象化三原则）+ inkos Writer（章节类型指导）+ tomato-novelist（情绪拐点+作者说）
> 使用方：Writer（Phase 1 Step 1 内化约束 + Step 2 写作时查阅）

---

## 一、6条句法红线

**红线1：最多1个比喻/页（约2000字不超3个比喻）**
- 坏："灵力如潮水般涌来，像春风般温暖，仿佛母亲的手"（3个比喻挤在一句）

**红线2：肯定句式为主，禁用双重否定**
- 坏："他并非不知道这样做的后果" → "他知道后果"
- 坏："不可否认的是" → 删除，直接写事实
- 坏："不是没有可能" → "有可能"

**红线3：短句+句号——动作场景禁用破折号接力**
- 坏："他冲过去——剑光闪过——血溅了出来——"
- 好："他冲过去。剑光闪过。血溅了出来。"
- 动作越激烈，句子越短。

**红线4：禁止全知形而上学总结**
- 坏："这就是命运的齿轮，在无人知晓的角落悄然转动"
- 坏："历史将记住这一天，虽然此刻他们还浑然不觉"
- 好：直接写发生了什么。不评价它的意义。让读者自己感受。

**红线5：中文语感——禁用西式从句堆叠**
- 坏："当他站在那个曾经见证无数修士来来去去如今已荒废的广场上时"
- 好："他站在广场上。这里曾见证无数修士的来去。如今荒废了。"
- 定语>15字 → 拆成独立的句子。

**红线6：剥离心理锚点——删除"他知道/感觉到/意识到"**
- 坏："主角知道这一次封印的碎裂比前两次更猛烈"
- 好："封印的碎裂比前两次更猛烈——剧痛从骨髓深处炸开来"
- 坏："他感觉到一股凉意爬上脊背" → 好："凉意爬上脊背"
- 规则：任何"X知道/Y感觉到/Z意识到" → 删除前三个字，直接写后面的内容。

## 二、反AI写作

**段落控制**：
- 30-50%段落为单句段（一句话自成一段——用于强调、节奏、冲击）
- 每段50-100字为佳
- 禁止连续3段长度相同（会产生机械节奏）

**句子控制**：
- 长短句交替：长句（25-40字）用于描写/内心/蓄力，短句（5-15字）用于动作/对话/爆发
- 禁止连续5句长度相同

**禁用词**：
- 核心禁止：仿佛、不禁、宛如、竟然、不知为何、莫名其妙地、陡然、刹那、浑身上下
- 心理锚点：他知道、他感觉到、他意识到、他想起（除非刻意安排的回忆）
- 形而上学：命运的齿轮、历史的洪流、无人知晓的角落、冥冥之中
- 冗余修饰：极致的、无比的、彻底的、疯狂地、剧烈地（除非真正达到了极致）

## 三、五感与推拉镜头

**推镜法则**：每个场景的第一段必须遵守——先切入微观感官，再拉出宏观环境。绝不一上来就上帝视角全景。
- 错误："这是一个阴暗的房间，墙上挂着剑。"
- 正确："铁锈味钻进鼻腔。他停下脚步，手指触到墙壁的潮湿——是血。墙上，一排剑静静悬挂。"
- 法则：第一句=声音/气味/触觉/温度 → 第二句=角色对感官的反应 → 第三句=从感官细节自然过渡到环境全景

**五感交替**：每章至少覆盖3种感官。
- 听觉：松涛/剑鸣/对话语调/沉默的重量——沉默也是听觉
- 触觉：松针刺痛/汗浸湿冷/石阶粗糙/灵力温热——触觉是最被忽视的沉浸工具
- 嗅觉：松脂/血腥/丹药苦香/泥土潮湿——嗅觉直接触发情绪记忆
- 味觉：铁锈味(血)/丹药苦涩/灵茶清甘
- 视觉：不言自明。但不要连续3段纯视觉描写。
- 检查：如果本章只有视觉→必须补充至少2种其他感官。

**场景开头 Push-In（推入）**：从微观感官细节→拉远到宏观场景。不要从概括句开始。
- 好："松针尖刺进掌心的刺痛。月光从破漏的屋顶漏下来，在地面投出银白斑块。守林人的小屋在松涛中静默。"

**场景结尾 Pull-Out（拉出）或 Hold（悬停）**：爆发章末→拉出到更广阔的威胁/悬念。过渡章末→悬停在当下的氛围中。

**对话占比**：30-50%为佳。<30%可能流水账严重，>50%可能像剧本缺乏沉浸。

## 四、具象化三原则

**原则1：时间具体化**
- 坏："过了一会儿" / "一段时间后" / "不久之后"
- 好："三炷香燃尽时" / "他从天亮等到日头偏西" / "茶凉了两次"

**原则2：人物具体化（数量具象）**
- 坏："一群修士围了上来" / "很多人"
- 好："七个修士围了上来——三个开元境，四个凝魂境"

**原则3：尺度具体化**
- 坏："大量灵石" / "巨额债务" / "极强的灵压"
- 好："三百枚灵石，这是他三年的积蓄" / "灵压沉重——连石阶都在发颤"
- 例外：角色对话中允许模糊表达（角色可以说"很大"——但叙述不能说）。

## 五、章节类型写作指导

**爆发章**：节奏快→更快→爆发→回落。前半段快速爬升紧张度。中段高潮——打脸/突破/反杀集中投放。后半段高潮后果+他人反应+章末钩子。句子短，动作密集，对话锋利。禁止在高潮前插入大段环境描写或内心独白。

**蓄压章**：节奏承接→积累→小进展。前半段承接前章钩子+布置本章目标。中段积累——获取资源/学习技能/搜集线索/埋新伏笔，允许细节渲染。后半段推到一个小进展。章末钩子必须让读者有翻下一章的理由。

**后效章**：节奏冷却→反应→新威胁苗头。前半段展示高潮后的世界变化。中段他人反应——核心事件聚拢多角色时，每人独立内心反应（圆心法同场多视角）。后半段新威胁的苗头。节奏可放缓，这是给读者的"呼吸章"。章末必须有"下一件事在靠近"的暗示。

**过渡章**：节奏最慢。每段都要有功能——世界质感展示/人物关系日常深化/支线进展/为后续伏笔铺路。章末可豁免强钩子——但要有余韵。稀疏 Memo 豁免：如果 Memo 内容少，不必强行加戏。

## 六、单章情绪拐点

2200-2800字内至少3次情绪变化：
- 开头（50-250字）：承接前章情绪或用一个反常细节打破预期
- 中间（1500-1900字）：情绪逐步升级，至少一次小起伏
- 结尾（450-650字）：情绪到达本章峰值后急转，为钩子服务

每3章完成一次"压-小扬-压-爆"循环。

### 情绪拐点句式库

来源：chapter-template-tomato.md。直接可用的情绪转折句式：

| 情绪转折 | 示例句 |
|---------|--------|
| 憋屈→暗爽 | "他低着头，所有人都以为他在认错——直到看见他嘴角压不住的笑。" |
| 震惊→愤怒 | "她的手在发抖。不是因为害怕——是因为看清了那个人是谁。" |
| 绝望→希望 | "牢门开了。不是狱卒——是那个他说过'永远不会来'的人。" |
| 得意→恐惧 | "他笑着伸手去接战利品——指尖碰到的一瞬间，笑容僵在了脸上。" |
| 平静→震惊 | "信上只有一行字。她读了三遍——每一遍都希望自己看错了。"

## 七、作者说格式

1-3句话。不剧透下一章。不乞求数据。可以：点明本章被回收的伏笔/预告下一章的情绪方向/补充一个本章的有趣细节。

好例子："封印碎裂的痛感是我写过最疼的场景。但主角没晕过去——不是因为意志力强，是因为三年的针刺痛楚已经把他的痛觉阈值抬到了不正常的高度。"

坏例子："下一章兽潮来袭，主角大展神威，废材逆袭正式开始！"（剧透+俗套）

## 八、黄金开篇专项（仅第1章）

**书架页核心认知**：封面 + 书名 + 简介前三行（约50字）。这50字决定50%的点击率。读者在书架页看到的就是这50字——它决定了读者是点进来还是划走。

- 输出3个版本的开篇50字（强冲突型/强反差型/强情绪型）
- 用筛选标准评估后选定最优版本
- 避雷针检查：天气/睡醒/照镜子/简历式/世界观/流水账 → 一律禁止

**十种强力开头**（来源：chapter-craft-guide.md）：
1. 行动开头 — 主角正在做某事，立即建立节奏感
2. 逆势开头 — 与预期相反的事件发生
3. 震惊对话 — 一句令人震惊的对话作为第一句
4. 倒计时开头 — 时间紧迫压力立现
5. 重大发现 — 主角发现改变一切的东西
6. 危机时刻 — 开场就在危机中
7. 谜题浮现 — 什么东西不对劲
8. 背叛开场 — 信任被打破
9. 重大选择 — 主角必须在两个选项中抉择
10. 结局预览 — 先把结果抛出来再倒叙

**致命错误（6种，禁止）**：天气描写开头/睡醒开头/照镜子开头/简历式人物介绍/世界观说明开头/日常流水账开头。

### 各题材开篇句式示例

来源：golden-opening-template.md。

| 题材类型 | 开篇示例 |
|---------|---------|
| 爽文打脸型 | "所有人都以为他是个废物——直到那道天雷劈下来的时候，他连躲都懒得躲。" |
| 悬疑反转型 | "尸检报告上的死因是自杀。但法医偷偷告诉我——尸体的手指全是反方向折断的。" |
| 虐文觉醒型 | "他等了十年。等到的是她挽着别人的手从他面前走过——连看都没看他一眼。那天晚上，他把那枚戒指扔进了河里。" |
| 马甲掉落型 | "我隐藏身份在宗门当了三年杂役。今天掌门的女儿当着所有人的面——喊了我一声师叔祖。" |
| 追妻火葬场型 | "离婚协议签完的那天，他头也没回地走了。三个月后，他在新闻上看到她——旁边站着一个比他强一百倍的男人。" |
| 反套路喜剧型 | "系统提示：您的外挂已到账。我打开一看——里面只有一行字：'自己努力。'" |
| 悬疑惊悚型 | "镜子里的我倒退了半步。但我本人——站在原地没动。" |
| 治愈甜宠型 | "他说要给我看个宝贝。我闭上眼等了半天——他把我的手按在他心跳上。'听到了吗？你的名字。'" |
| 脑洞大开型 | "地球被归类为'未开化星球'的第3721年，一个人类通过了星际高考。考场炸了。" |

## 九、中文文学五大技法

来源：chapter-craft-guide.md。

1. **白描**：用最少的文字，写出最多的信息。不用形容词描写，用动作和对话让读者自己构建画面。
2. **留白**：重要的情节转折/情感爆发点前留出空白。给读者想象空间——不把每个细节都写完。
3. **意象营造**：用具体物体承载抽象情感。不是"她很孤独"，而是"她每天擦一遍那盏早已没有油的灯"。
4. **草蛇灰线**：重要线索分多次、递减式暗示。第一次明显，第二次隐晦，第三次只留一丝痕迹——让读者回看时恍然大悟。
5. **蒙太奇剪辑**：时间和空间跳跃拼接，用一个共同的意象或情绪作为转场轴。看似无关的场景并置产生新的意义。

## 十、爽文执行公式

来源：satisfaction-formulas.md。

### 打脸公式（7步）
1. 质疑 → 2. 嘲讽升级 → 3. 展示 → 4. 初步震惊 → 5. 真相揭示 → 6. 打脸 → 7. 主角淡定

**密度控制**：爆发章2-3次 / 蓄压章1次 / 过渡章0-1次。禁用模式：重复打脸（连续2章同模式）、无敌打脸（对手太弱）。
**打脸间隔**：每3-5章一次完整打脸。

### 装逼公式（5步）
1. 低调隐藏 → 2. 被迫出手 → 3. 一招制敌 → 4. 众人震惊 → 5. 淡定离开

主角的淡定比任何台词都有力量。与打脸的区别：打脸需要逐步升级的冲突，装逼是一锤定音。

### 升级公式（6步）
1. 遇强敌 → 2. 压力巨大 → 3. 顿悟/机缘 → 4. 突破 → 5. 实力展示 → 6. 反杀

**频率控制**：小突破每3-5章 / 中突破每10-15章 / 大突破每30-50章（卷级）。禁止战斗中临时觉醒新能力（除非此前≥5章埋了伏笔）。每次突破后必须有适应期（≥1章）。

## 十一、金句锻造

来源：golden-lines-guide.md。

**四种金句类型**：
1. **爆发句** — 情绪至极的短句。去掉所有修饰，只留最核心的主谓。用句号制造定格。例："他等这一天。等了十年。"
2. **揭示句** — 真相揭露的瞬间。不是解释，是展示。例："那个符文——不是封印。是钥匙。"
3. **情感句** — 关系转折的浓缩。不写"他们相爱了"，写一瞬间的动作或对话。例："她转身时，他第一次没有叫她师妹。"
4. **定格句** — 章节或场景的最后一句，制造余韵。用具体意象收束。例："窗外，雪还在下。"

**锻造时机**：爆发章3-5句 / 蓄压章1-2句 / 后效章1-2句 / 过渡章0-1句。

**禁忌**：不用四字成语、不在金句后加解释、不用感叹号、不重复同一句式。

## 十二、对话写作

来源：dialogue-guide.md。

**每句对话至少完成之一**：推动情节/揭示人物/制造冲突/传达信息/表达情感/制造悬念。

**潜台词技法**：
| 技法 | 说明 |
|------|------|
| 话题转移 | 不回答上一个问题——转移本身就暴露了回避 |
| 反问而非回答 | "你觉得呢？"——把球踢回去 |
| 谈论其他事物 | 不说正事——绕圈子本身就是态度 |
| 沉默和动作 | 不说话——动作或沉默代替回应 |

**权力博弈**：高位者提问/打断/控制话题/使用沉默；低位者回答/被打断/跟随话题/打破沉默。

**四类对话场景**：争吵（节奏快/句子短/潜台词少/直击要害）、告白（节奏慢/句子短但停顿多/动作穿插多）、暧昧（节奏慢/句子绕/潜台词多/眼神代替话）、告别（节奏由慢变快/最后一句短/定格）。

## 十三、内容扩充技巧

来源：content-expansion.md。字数不足时使用。

1. **关键时刻放慢**：战斗/告白/揭秘的关键瞬间，把1秒掰成5秒写——动作分解+感官+内心反应
2. **内心世界展开**：思维链（主角在想什么→为什么这么想→这和之前有什么关联）/ 感官触发记忆 / 内心对话 / 身体反应映射
3. **对话层次丰富**：不只是对话本身——加动作表情打断/潜台词层/话题绕弯/权力博弈
4. **次要情节穿插**：配角小故事/暗线一笔推进/伏笔自然埋设/关系微妙变化

## 十四、叙事技巧

来源：narrative-techniques.md。

**倒叙**：触发点必须是本章的感官细节（不是"他想起了三年前..."）。回到当前时间时要有一个对应的感官细节呼应。

**多视角**：每章保持一个POV。如必须切换POV，一整章用一个POV，章首声明。

**不可靠叙述者**（3种）：
1. 刻意隐瞒型 — 叙述者知道真相但不对读者说实话
2. 认知偏差型 — 叙述者自己也没搞清楚真相
3. 记忆错误型 — 叙述者的记忆本身就是错的

**经典叙事模式**：
- 起承转合：四段式，每段约25%
- 平行蒙太奇：A线-B线-A+B线，两条线在结尾交汇
- 倒计时结构：章首设定截止时间→中间行动→末尾结果+钩子

## 十五、特殊章节场景序列

来源：special-chapter-templates.md。

**战斗章节**：战前20%（对峙/环境/策略）→ 战中60%（攻防交替/关键转折/高潮一击）→ 战后20%（结果/代价/反应）。禁止战斗中长篇内心独白，禁止一招秒杀（除非装逼场景）。

**突破章节**：突破前25%（积累/征兆/准备）→ 突破中50%（感官分解——经脉/灵力/识海/肉身逐一展开）→ 突破后25%（验证新能力/适应期/代价）。禁止战斗中突然觉醒，禁止意志力爆发送跨境。

**转折章节**：铺垫30%（日常中的异常）→ 异常积累30%（越来越多的不对劲）→ 揭示15%（真相揭露）→ 冲击25%（后果+反应+新方向）。揭示不能靠角色突然想起来或恰好发现。

## 十六、反AI写作补充规则

来源：deep-polish-rules.md。

**形容词硬性压缩**：一个名词最多配1个形容词。检测方法：搜索"的"——如果连续出现3个"的"，必须改写。

**禁用词**（按优先级）：
- RED（必须替换）：仿佛、不禁、宛如、竟然、不知为何、莫名其妙地
- ORANGE（优先替换）：陡然、刹那、浑身上下、心中暗想、像某种...
- YELLOW（尽量替换）：然而、极其、无比、彻底地、疯狂地、剧烈地

**润色禁区**：不改情节/不改台词含义/不增删人物事件/字数变化≤±30字。

**扩展禁用词列表**（来源：deep-polish-rules.md）：
- RED 追加：在...的过程中、不仅...而且...、总而言之、值得注意的是、某种程度上、唯一的...到最后、弥漫、摇摇欲坠
- 追加规则：不是(用于否定状态)→改写为肯定句式；没有(用于否定状态)→描述存在本身；像某种...→删除；心中暗想→删除
- 璀璨/瑰丽/绚烂/绝美/无与伦比/无可挑剔/完美无瑕 → 替换为具体描写

**四字格律堆砌规则**：连续出现 ≥2 个四字成语/格律 → 保留最必要的一个，其余拆解为具体描写。

**抽象陈述转具体**：
- "他的实力很强" → 直接写他做了什么让旁人震惊的事
- "气氛紧张" → 写具体的人的动作/呼吸/眼神
- "她很美" → 写看到她的人的反应

**段落结构禁令**：禁止连续3段以相同句式开头。禁止连续3段以对话开头。

## 十七、打破读者预期四技法

来源：chapter-craft-guide.md。

1. **预期反转**：先建立读者对一个角色/事件/规则的预期，然后在关键节点反转——但反转必须有前文铺垫，不能凭空出现
2. **信息差利用**：让读者知道而角色不知道（戏剧反讽），或角色知道而读者不知道（悬念），利用这种不对等制造张力
3. **复杂动机**：角色的行为——特别是反派——不止一个动机。当读者以为理解了动机时，揭示更深层的一个
4. **节外生枝**：在解决一个问题的过程中制造新的问题，让节奏不要"解决即结束"

## 十八、信息密度与题材节奏

来源：chapter-craft-guide.md + narrative-techniques.md。

**信息密度波浪**：每章应有2-3个密度波浪。高密度(动作/对话/揭示) → 低密度(沉淀/描写/内心) → 高密度(新冲突/转折) → 低密度(情感消化/环境过渡)。连续3段以上高密度=读者疲劳；连续3段以上低密度=读者无聊。

**张弛有度原则**：
- 连续快节奏>3章 → 读者疲劳
- 连续慢节奏>2章 → 读者弃书
- 过渡章应在快节奏章之后
- 爆发章前必须有1-2章蓄压

**节奏控制（按句长和段落）**：

| 节奏 | 句长 | 段长 | 适用 |
|------|------|------|------|
| 快 | 5-15字 | 单句成段 | 战斗/追逐/紧张对峙 |
| 中 | 15-30字 | 2-4句/段 | 对话/日常/过渡 |
| 慢 | 30-50字 | 3-6句/段 | 心理独白/环境渲染/情感场景 |

## 二十、对话排毒与常见问题

来源：deep-polish-rules.md + dialogue-guide.md。

**端水发言检测**：删除角色名——如果这句话任何角色都可以说，就是端水发言，必须重写。每句对话必须带有该角色的语言指纹。

**常见问题**：
1. 信息倾倒 — 角色在对话中大段解释设定。对话不是说明书
2. 所有人说话一样 — 老谋士和小师妹用同样的句式
3. 过度礼貌 — "请问""麻烦""谢谢您"充斥对话，现实中的紧迫场景不会这样
4. 说教 — 角色突然变成作者的传声筒

**沉默的力量**：
- 拒绝回答 → 暗示隐瞒或保护
- 反应延迟 → 暗示震惊或权衡
- 对话中断 → 暗示情感崩溃或决心
- 以动作代替回答 → 比语言更有力

## 二十一、场景肌理充实（内容扩充技一）

来源：content-expansion.md。

场景太薄时，从三个维度同时展开：
1. **感官层**：视觉（光线/颜色/质感）+ 听觉（环境音/对话语调）+ 嗅觉（气味）+ 触觉（温度/质地）
2. **空间层**：近景（角色身边）→ 中景（房间/场景）→ 远景（窗外/远处），三个距离交替出现
3. **情绪层**：环境氛围不是客观描写——是通过角色的情绪滤镜看到的。愤怒时看什么都是刺眼的，悲伤时看什么都是灰的

**诊断-扩充实操流程**：
1. 诊断 → 场景哪个维度最薄？感官/空间/情绪？
2. 选择技巧 → 从扩充技巧中选1-2个最适合本章的
3. 找到插入点 → 不要在紧张对话中间插环境描写——在动作间隙、对话停顿处
4. 检验 → 删除扩充部分后故事是否变差？如果没变差→扩充是注水

## 二十二、金句补充

来源：golden-lines-guide.md。

**金句的本质**：在情绪最高点释放的一句话，让读者产生生理反应——瞳孔放大、心跳加速、呼吸暂停、想去评论区打"啊啊啊啊"。

**金句 vs 普通好句**：
- 普通好句：读者点头，"写得不错"
- 金句：读者停下，重读一遍，截图发朋友圈

**各题材金句风格**：

| 题材 | 风格 | 示例方向 |
|------|------|---------|
| 玄幻爽文 | 冷而有力，短促如剑 | "这一剑。十年的账。" |
| 都市悬疑 | 信息量爆炸，看似平淡实则惊雷 | "那份文件的日期——是明天。" |
| 仙侠言情 | 克制中爆裂，一句胜千言 | "他没有回头。但脚步慢了半步。" |
| 系统流/脑洞 | 规则反转，认知颠覆 | "系统提示：你的外挂已过期。请续费。" |
| 种田 | 平淡中见深情，日常中见非凡 | "今年的收成。够两个人吃了。" |
| 无限流 | 绝望中的黑色幽默 | "恭喜你通过了新手关。——新手关共99层。" |

## 二十三、特殊章节补充规则

来源：special-chapter-templates.md。

**战斗章节**：战斗中的对话——短、挑衅或惊慌、不超过10字。

**突破章节**：突破中的感官分解按类型展开——触觉（身体变化）/ 视觉（内视经脉/灵力流动）/ 听觉（体内声音）/ 外界反应（天地异象/旁人感知）。突破后用"第一个动作"展示新境界——不写"他现在是金丹期了"，写他下意识做了一件之前做不到的事。

**转折章节**：揭示的信息必须在铺垫阶段有≥2个可回溯的线索——读者回看时能发现"原来那个细节是这个意思"。不能靠角色突然想起来或恰好发现。

## 二十六、开篇筛选标准

来源：golden-opening-template.md。选定3版本开篇后逐项检查：

- □ 50字内出现主角？
- □ 包含反常点或冲突点？
- □ 读者3秒内产生"然后呢？"追问欲？
- □ 符合本章情绪标签？

### 开篇自检清单

- □ 前10个字出现主角（非天气/环境/路人）
- □ 包含反常或冲突点（不是"日常的一天"）
- □ 3秒感知情绪（爽/虐/悬疑/甜蜜——读者立即知道这是什么类型的故事）
- □ 避免6种避雷针（天气/睡醒/照镜子/简历式/世界观/流水账）
- □ 如果这是书架页展示的50字——你会点击吗？

### 开篇避雷针（详细版）

| 错误类型 | 错误示例 | 致命原因 |
|---------|---------|---------|
| 天气描写开头 | "天空阴沉沉的..." | 读者不关心天气——关心发生了什么 |
| 睡醒开头 | "他睁开眼..." | 最陈词滥调的开篇 |
| 照镜子开头 | "他站在镜子前..." | 简历式人物介绍的开端 |
| 简历式人物介绍 | "他叫XX，今年XX岁，是XX宗门的XX弟子..." | 信息倾倒——不是故事 |
| 世界观说明开头 | "这个世界分为三大洲..." | 用故事展示世界观——不是说明书 |
| 日常流水账开头 | "他早上起床、刷牙、吃早饭..." | 除非早饭里有毒——否则删掉 |
