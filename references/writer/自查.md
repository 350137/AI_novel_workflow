# 写后自检清单

> 来源：quality-checklist.md + writer-self-check.md（合并去重）
> 使用方：Writer（Phase 1 Step 3 写后自检，输出前必做）

---

## 一、字数检查
- □ 正文（不含作者说和追踪卡）是否在 Genre Profile chapter.word_count 范围内？
- □ 不足 → 优先环境渲染/内心独白/动作分解补足
- □ 超出 → 检查是否有注水段

## 二、句法红线
- □ 比喻数 ≤ 3个？
- □ 搜索"没有不|不可否认|并非不|不是没有"→清空
- □ 动作场景搜索"——"→改成句号
- □ 搜索"命运的|历史的|无人知晓|冥冥"→清空
- □ 搜索"当...的时候"长句→拆分
- □ 搜索"知道|感觉到|意识到|不禁|仿佛|宛如"→清空或改写

## 三、具象化
- □ 搜索"过了一会儿|不久之后|一段时间后"→改写为可感知的时间流逝
- □ 搜索"一群|很多人|不少人"→计数或具象化
- □ 搜索"大量|巨额|极强|非常|十分|极其"→替换为可感知后果或精确数字

## 四、感官与节奏
- □ 覆盖了≥3种感官？连续3段纯视觉？
- □ 对话占比30-50%？
- □ 30-50%段落为单句段？没有连续3段长度相同？

## 五、章末钩子
- □ 过渡章豁免。非过渡章→章末有钩子？钩子类型与 Memo 一致？
- □ 钩子强度与章位匹配？（过渡章≤2，爆发前章≥3，卷末≥4）
- □ 最近3章结尾句式/意象无重复？

## 六、Memo 遵守
- □ 兑现项全部满足？
- □ 扣留项一个都没写？
- □ 禁止事项全部遵守？
- □ PRE_WRITE_CHECK 全部落地？

## 七、情绪与开篇
- □ 本章有≥3次情绪变化？
- □ 情绪峰值在中后段（不在开头）？
- □ 前3段抓住读者？（行动/冲突/悬念——非天气/日常/回顾）
- □ 与上一章有连接？背景信息不过量？

## 八、钩子账本
- □ resolve N → open ≥ N？
- □ 如果 open < resolve → 在 writerNotes.deviations 中标注原因

## 九、评分卡（8维×10分，>60可提交，>70优秀）

| 维度 | 分数(0-10) |
|------|-----------|
| 基础要素（字数/格式/钩子存在） |  |
| 开头质量（前三段抓住读者） |  |
| 句法红线（6条全通过） |  |
| 反AI写作（无禁用词/句式） |  |
| 具象化（时间/人物/尺度） |  |
| 对话（占比/自然度/标签） |  |
| 感官（≥3种/推拉镜头） |  |
| 钩子质量（力度/类型/承接） |  |
| **总分** | __/80 |

任一维度<5 → 标注在 writerNotes.continuityConcerns 中，提交前修复。

## 十、穿帮自检

来源：writer-self-check.md。

### 角色状态穿帮
- □ 角色此刻的境界/能力是否与前章一致？（不能突然用出没学过的技能）
- □ 伤势进展是否合理？（不能"断了肋骨→下章活蹦乱跳"）
- □ 角色位置是否连贯？（不能"城东→城西"无移动过程）
- □ 持有物品是否与已获得的一致？（不能"突然掏出一把没提过的剑"）

### 地点穿帮
- □ 本章场景特征与前文描写一致？（同一地点不能突然"多了一扇窗"）
- □ 场景间距合理？（不能"矿洞→宗门内场"零距离瞬移）

### 能力/等级穿帮
- □ 本章写角色使用某能力→确认他已获得
- □ 本章写跨境战斗→满足≥2项条件（功法克制/灵气环境/神通针对性/法宝碾压/战术信息差）
- □ 本章写突破→前3章有积累描写？突破后预留适应期？
- □ 金手指触发条件→本章事件确实满足条件？

### 时间线穿帮
- □ 本章事件与前后章时间衔接正确？
- □ 如有时间跳跃→标注了跳跃量？
- □ 场景之间时间连贯？（不能"中午→深夜"无过渡）

## 十一、爽点自检

来源：writer-self-check.md。

- □ 本章包含≥1个爽点推进？（非过渡章必须）
- □ 爽点类型与前2章轮换？（不连续3章同类型）
- □ 爽点是否"完整"？（打脸需含：被小看→展示实力→他人反应三环节）
- □ 过渡章→至少1个微型爽点（幽默互动/认知小颠覆/关系微进展）
- □ 本章爽点是否与 Memo §2（兑现）对齐？

---

## 十二、禁忌自检

来源：writer-self-check.md。

- □ 疲劳词：搜索 Genre Profile fatigue_words → 同词<3次？
- □ 禁用模式：搜索 Genre Profile forbidden_patterns → 零命中？
- □ 世界铁律：本章情节是否违反 Genre Profile world_rules.core_laws？
- □ 机械降神：是否有任何"恰好""刚好""突然发现"式巧合？
- □ 反派降智：反派是否为了衬托主角而做了不合理决策？
- □ 角色消失：是否有配角出场后>15章未再出现？

---

## 十三、对话专项检查

- □ 对话标签不重复（"说"重复率<30%）？
- □ 不同角色说话腔调不同？（删除角色名后仍能分辨是谁在说话）
- □ 高情绪场景中对话缩短？（紧张/愤怒时句子变短）
- □ 每句对话有目的？（推进/揭示/冲突/传达/情感/悬念——至少之一）
- □ 搜索"的的的"连续 → 改写
- □ 对话中是否有潜台词？（未直接说出的真正意思）
- □ 是否有沉默或动作打断长对话？（沉默也是一种回答）

## 十四、钩子追问与金手指检查

- □ 章末钩子是否能让读者产生"然后呢？"的追问欲？（不只是"存在钩子"，而是"读者有翻下一章的冲动"）
- □ 如有金手指使用 → 是否满足 Genre Profile golden_finger.trigger_condition？
- □ 本章是否包含"作者说 + 隐藏情节追踪卡"？（格式完整性检查）
