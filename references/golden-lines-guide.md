# 金句锻造指南

> 来源：tomato-novelist 金句锻造（情绪高潮处加入可划线评论的金句）
> 用途：Writer 和 Polisher 在情绪高潮处锻造金句。这些句子会被读者划线、评论、分享。

---

## 一、什么是金句

金句不是华丽的修辞。金句是**在情绪最高点释放的一句话，让读者产生生理反应**——瞳孔放大、心跳加速、呼吸暂停、想去评论区打"啊啊啊啊"。

### 金句 vs 普通好句

| 普通好句 | 金句 |
|---------|------|
| "她的剑很快。" | "她出手了。然后战斗就结束了。" |
| "他等了三年。" | "三年。一千零九十五天。他等的不是复仇——是道歉。" |
| "他发现自己被骗了。" | "原来从头到尾，他都是那个被利用的人。" |
| "两人终于在一起了。" | "她等他等了九世。这一世，换他等。" |

**区别**：金句有"定格感"——读者读到这里会停下来，呼吸变浅，眼睛在句子上多停 0.5 秒。

## 二、金句的四种类型

### 2.1 爆发句（战斗/对决高潮）

**特征**：极端简洁。主语+动词。句号。不多解释。

```
"她出手了。"
"他输了。"
"门开了。"
"她还活着。"
```

**锻造方法**：去掉所有修饰，只留最核心的主谓。用句号制造定格。

### 2.2 揭示句（真相揭露）

**特征**：一句话推翻读者之前的全部假设。用"原来"、"不是...而是..."、"从来都是"。

```
"原来从头到尾，最信任的人才是最想杀她的人。"
"封印不是意外——是献祭。"
"他不是在修炼。他是在被消化。"
```

**锻造方法**：先建立读者预期，然后用金句推翻。金句之前的段落是铺垫——金句是引爆点。

### 2.3 情感句（关系转折）

**特征**：克制。不是"我爱你"——而是用具体细节传达情感。

```
"他没回头。他知道一回头就走不了了。"
"她的手指碰到了他的——然后收了回去。"
"他等了这么久的话，真的听到了，反而不知该怎么回答。"
```

**锻造方法**：写角色的犹豫、退缩、微小的动作。最浓的情感用最淡的笔写。

### 2.4 定格句（章节终结）

**特征**：在章末最后一句话制造"时间暂停"感。

```
"然后门铃响了。"
"屏幕上弹出了一条消息——来源：未知。"
"她笑了起来。这是本章最后一句话。但下一章的第一句话是：'她笑错了。'"
```

**锻造方法**：用短句。制造信息断层（读者知道的<角色不知道的，或反之）。

## 三、金句锻造时间点

| 章节类型 | 至少锻造 | 最佳位置 |
|---------|:------:|---------|
| 爆发章 | 3-5 句 | 战斗高潮、突破时刻、战后反应 |
| 蓄压章 | 1-2 句 | 压抑最深处、转折临界点 |
| 后效章 | 1-2 句 | 反应最强烈的时刻、关系变化瞬间 |
| 过渡章 | 0-1 句 | 如有世界观揭示或小反转 |

---

## 四、金句锻造禁忌

```
- ❌ 不能用四字成语堆砌（"他势如破竹、所向披靡、一往无前"——这是AI腔）
- ❌ 不能解释（"她出手了。她的剑很快，快得对方根本来不及反应。"——后半句毁了前半句）
- ❌ 不能用感叹号（"她出手了！"—感叹号出卖了不自信）
- ❌ 不能重复（同一章内不能有两句结构相同的金句）
- ❌ 不能偏离人物设定（冷峻主角的金句不会超过15字）
- ✅ 金句是可以被截屏的。写完问自己：读者会截图发微博/朋友圈吗？
```

## 五、各题材金句风格

| 题材 | 金句风格 | 示例 |
|------|---------|------|
| 玄幻爽文 | 冷+燃 | "他等了三年。等的就是这一天。" |
| 都市悬疑 | 冷+毛骨悚然 | "法医报告说死者死于凌晨2点。但她凌晨3点还在发朋友圈。" |
| 仙侠言情 | 虐+克制 | "三世了。这一世我不要你等——我要你忘了我。" |
| 系统流/脑洞 | 信息差+颠覆 | "系统提示：恭喜宿主获得999级。全服只有1级。" |
| 种田 | 收获+温暖 | "第一颗种子发芽的那天，她在这个世界第一次笑了。" |
| 无限流 | 规则反转 | "副本规则第3条：不要相信副本规则。" |
