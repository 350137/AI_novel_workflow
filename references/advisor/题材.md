# 题材分类与自适应提问

> 来源：fiction-crafter + tomato-novelist + novel-writer-cn
> 使用方：Advisor (Phase 1 收敛提问)

---

## 一、Q0 题材锚定

**Q0 必问内容**：
- **主类型**：玄幻 / 都市 / 仙侠 / 科幻 / 历史 / 悬疑 / 武侠 / 军事 / 奇幻 / 灵异 / 游戏 / 竞技 / 种田 / 无限流 / 系统流
- **子类型/融合标签**：如"玄幻+悬疑"、"都市+修真"、"仙侠+虐恋"——可以多选
- **风格基调**：纯爽文 / 爽中有虐 / 正剧 / 轻松搞笑 / 暗黑 / 文艺 / 热血燃
- **预期篇幅**：短篇<100章 / 中篇100-300章 / 长篇300-800章 / 超长篇800+章

**Q0 目的**：选定题材后，Q1-Q6 的问题内容和追问方向将根据题材动态调整。

## 二、题材自适应规则表

| 题材 | Q1 侧重 | Q2 侧重 | Q3 侧重 | Q4 侧重 | Q5 默认结构 | Q6 对标锚点 |
|------|---------|---------|---------|---------|------------|------------|
| 玄幻 | 世界观+力量体系 | 打脸/突破节奏 | 情感线辅助定位 | 扮猪吃虎/废材逆袭 | 爽文模型 | 起点玄幻榜 |
| 都市 | 现实规则+金手指 | 重生/商战/神医 | 情感线可为主线 | 重生先知/信息差 | 混合模型 | 都市分类热榜 |
| 仙侠 | 天道规则+因果 | 突破+情感双线 | 情感线通常重要 | 身份反转/三世因果 | 传统或混合 | 仙侠经典 |
| 科幻 | 科技设定+宇宙观 | 探索/生存紧张感 | 通常辅助或无线 | 降维打击/知识碾压 | 传统模型 | 硬科幻/软科幻 |
| 历史 | 朝代规则+历史锚点 | 权谋/种田节奏 | 视朝堂/后宫而定 | 现代知识降维 | 传统或爽文 | 历史分类+正史 |
| 悬疑 | 谜题设计+信息控制 | 层层揭秘/反转 | 通常辅助 | 隐瞒真相/多重身份 | 传统模型（严格） | 本格/社会派 |
| 种田 | 世界规则轻量 | 收获/建设慢节奏 | 情感线常用 | 平凡中的非凡 | 传统模型 | 种田文经典 |
| 无限流 | 副本设计+规则 | 生存+突破双紧张 | 通常辅助 | 副本中身份切换 | 爽文+传统混合 | 无限流代表 |
| 系统流 | 系统规则+数值 | 升级/打脸/数据 | 视情况 | 系统附身/反差 | 爽文模型 | 系统流热榜 |
| 奇幻 | 魔法体系+种族 | 探险/战斗密度 | 视世界观 | 种族/血统反差 | 传统模型 | 西幻/东方奇幻 |
| 灵异 | 恐怖规则+阴阳 | 恐怖/惊悚节奏 | 通常辅助或无线 | 阴阳眼/通灵 | 传统模型 | 灵异/惊悚榜 |

**追问原则**：跳过不适用的追问。玄幻跳过"副本设计"，悬疑跳过"战力体系"，种田跳过复杂的规则追问。

**好追问 vs 差追问示例**：
- 好追问（玄幻）："你说'废材逆袭'，废材的原因是什么——天生废脉、被封印、还是被夺走天赋？"
- 好追问（悬疑）："谜题的公平性怎么保证——读者在揭晓前能自己推理出来吗？"
- 差追问："你想写什么样的故事？"（太宽泛，用户已说过）
- 差追问："你确定吗？"（不能引出细节）

**Q3 题材自适应默认倾向**（用户可覆盖）：
- 玄幻/科幻 → 默认倾向"辅助"但须确认
- 仙侠/都市 → 倾向"重要副线"
- 种田/甜宠 → 倾向"主线"

## 三、情绪标签与节奏选择

8 种核心情绪体验 + 1 种特殊（单选，混合需标注主次）：

| 类型 | 特征 | 节奏 |
|------|------|------|
| **爽文** | 打脸逆袭/扮猪吃虎/降维打击 | 快节奏，3章循环，句短，打脸密集 |
| **虐恋** | 追妻火葬场/虐主成长 | 先抑后扬，压抑期>5章须有"觉醒时刻" |
| **甜宠** | 日常温馨/双向奔赴 | 轻松稳定，爽点=治愈感+互动甜蜜 |
| **悬疑** | 层层揭秘/烧脑反转 | 信息控制，1-2章给线索不给全 |
| **爆笑反套路** | 沙雕玩梗/荒诞流 | 快节奏吐槽，爽点=预期打破+荒诞解读 |
| **热血** | 伙伴羁绊/燃向战斗 | 蓄压=训练/组队，爆发=战斗/牺牲/突破 |
| **惊悚** | 恐惧驱动 | 压抑感持续积累，爽点来自"劫后余生" |
| **温馨治愈** | 种田/慢生活 | 节奏舒缓，爽点=渐进收获+日常温暖 |
| **脑洞大开** | 系统流/末日囤货/星际经营/规则怪谈 | 爽点=信息差颠覆 / 规则反转，每1-2章一个新设定揭晓 |

**体裁特定追问**（仅当相关时）：
- 战斗题材 → "战斗密度如何？打脸间隔是1-2章还是3-5章？"
- 虐恋 → "虐持续多久给第一个糖？多少章前不能弃书？"
- 悬疑 → "反转频率？每卷几个核心反转？"

### 言情专项：情感节奏点与关系弧线

来源：genre-knowledge-summary.md（言情核心公式）。

**情感节奏点（必备，按顺序）**：

| 节奏点 | 时机 | 功能 |
|--------|------|------|
| 初次相遇 | 前 10-15% | 建立第一印象，埋下吸引力种子 |
| 障碍建立 | 20-30% | 为什么不能在一起？（外部/内部障碍） |
| 亲密加深 | 30-50% | 被迫共处/第一次合作/情感突破 |
| 关系危机 | 50-70% | 误会/背叛/秘密暴露——几乎分手 |
| 黑暗时刻 | 70-85% | 最低点——似乎不可能在一起了 |
| 大团圆（HEA） | 90-100% | 克服障碍，情感满足收束 |

**校验问题**：
- 删除关系线后，故事还成立吗？（应该不成立——关系线是主线）
- 读者更关心"他们会在一起吗"而非"发生了什么"？

### 悬疑专项：公平游戏原则与线索系统

来源：genre-knowledge-summary.md（Ronald Knox 十诫精简版）。

**公平游戏原则**：
1. 凶手必须在故事前期（前 30%）出场
2. 不可使用超自然力量解决谜题——真相必须是自然的
3. 不可用意外或直觉解决谜题——必须通过逻辑推理
4. 侦探必须公开所有发现的线索——读者拥有同等信息
5. 答案不能是"双胞胎"或"梦"（除非有充分铺垫）
6. 可以有红鲱鱼（误导线索）但必须在真相揭晓时有合理解释

**线索系统设计**：
- □ 核心线索：3-5 条，直接指向真相
- □ 红鲱鱼：2-3 条，指向错误答案但合理解释
- □ 每条线索在正文中出现时，至少一个角色注意到但不一定正确解读
- □ 真相揭晓时，所有线索必须能回溯验证
- □ 不能让主角"恰好"发现线索——必须有逻辑链条

### 复仇专项：仇恨强度与四阶段节奏

来源：genre-knowledge-summary.md（复仇公式）。

**仇恨强度公式**：仇恨强度 = 伤害程度 × 无辜程度 × 无法原谅度

**复仇四阶段**：
1. 受压——仇人势大，主角弱小，压抑感积累
2. 积蓄——暗中获取力量/证据/盟友，让读者看到逆袭希望
3. 反击——逐个击破仇人势力，爽点密集释放
4. 终局——直面最终仇人，最高潮

### 历史专项：大事不虚，小事不拘

来源：genre-knowledge-summary.md。

**大事（不可虚构）**：重大历史事件的时间、结果不可改变；著名历史人物的生卒年、重大功绩、历史定位不可改变；时代大背景（社会制度/科技水平/地理）不可改变。

**小事（可以虚构）**：虚构的小人物、历史缝隙中的日常生活、合理的细节填充、史书未记载的"可能性"。

**时代氛围检查**：
- □ 角色的思想观念是否符合时代？
- □ 物品/服饰/称谓是否正确？
- □ 社会阶层是否合理？
- □ 语言是否有时代感？
- □ 日常生活细节是否准确？

## 四、作品基因（防跑偏锚点）

来源：outline-template-tomato.md。Advisor 方案定稿后，提炼以下5个字段作为整本书的"防跑偏锚点"。后续所有 Agent 产出都应与此对齐。

| 基因字段 | 说明 | 示例 |
|---------|------|------|
| 情绪标签 | 核心情绪体验（单选从8+1种中选择） | 爽文 |
| 一句话卖点 | 抖音标题体——3秒传播 | "被封印的废材其实是万年前布下的棋子" |
| 主角一句话人设 | [身份]+[反差点]+[欲望] | "15岁杂役+前世准帝转世+打破帝位垄断" |
| 黄金开篇50字 | 书架页展示的前50字 | （见黄金开篇专项） |
| 终极钩子 | 读者读完最后一章得到的具体画面或台词 | "他在帝位上坐下——然后废除了帝位独占制" |

## 五、核心反差模式（原四）

**通用反差模式**（6种）：
- 扮猪吃虎（表面平凡实则强大）
- 马甲掉落（多重身份逐一揭晓）
- 降维打击（高维知识/技能碾压低维世界）
- 重生复仇（前世记忆+今生逆袭）
- 信息差碾压（系统/攻略/预知/未来记忆）
- 身份反转（以为是小人物，实则是传说中的XXX）

**题材特定反差**：
- 玄幻/仙侠 → 被封印者、废材逆袭、天才陨落再起
- 都市 → 隐世高手、重生经商、神医归来
- 悬疑 → 失忆侦探、双面人生、卧底身份
- 科幻 → 来自低维的高维生物、时间穿越者
- 种田 → 普通人的不普通坚持、平凡中的传奇

## 六、武侠专项

来源：genre-knowledge-summary.md。

**核心公式**：武侠小说 = 江湖世界观 + 武功体系 + 侠义精神 + 恩怨情仇

**江湖规则**：白道（名门正派）/ 黑道（魔教、邪派）/ 灰色地带（独行侠、隐世高人）/ 江湖规矩（以武会友、恩怨分明、长幼有序、不杀无辜）

**武功体系**：

| 层次 | 水平 | 代表人物 |
|------|------|---------|
| 三流 | 普通江湖人 | 镖师、普通弟子 |
| 二流 | 门派精英 | 首席弟子、护法 |
| 一流 | 掌门/长老级 | 各派掌门 |
| 绝顶 | 武林神话 | 传说级高手 |

**武学制约**：没有无敌的武功——相生相克 / 天赋+努力+机缘=成长公式 / 内力不是无限的——消耗必须合理 / 绝招有代价——使用后虚弱/反噬/冷却

## 七、标题创作

来源：title-creation-guide.md。

**题材-风格映射**：

| 题材 | 标题风格 | 示例 |
|------|---------|------|
| 玄幻 | 霸气/神秘 | 《万古第一神》《剑来》 |
| 都市 | 现实/反转 | 《重生之都市修仙》《隐世高手在都市》 |
| 悬疑 | 悬念/冷峻 | 《十日终焉》《心理罪》 |
| 言情 | 情感/诗意 | 《你是我的荣耀》《偷偷藏不住》 |
| 历史 | 厚重/史诗 | 《庆余年》《琅琊榜》 |
| 科幻 | 概念/冲击 | 《三体》《流浪地球》 |
| 无限流 | 规则/悬念 | 《惊悚乐园》《全球高考》 |

**标题创作五技法**：
1. 核心冲突提炼法：把全书核心矛盾浓缩为标题
2. 主角命名法：主角名+核心标签
3. 意象隐喻法：用一个意象暗示故事核心
4. 反差法：两个矛盾概念的并置
5. 悬念留白法：只说一半，激发好奇

**质量标准**：必须满足——3秒内产生追问欲 / 符合题材风格 / 不剧透核心反转。必须避免——太长的标题 / 纯关键词堆砌 / 陈词滥调。

## 八、灵感库

来源：inspiration-library.md。

**角色名**：林风、苏寒、叶尘、秦霜、白夜、墨渊、云曦、洛清、萧然、沈默、陆离、江月、楚河、魏来、方寻

**性格特质**：冷峻寡言、贱萌吐槽、温柔腹黑、热血冲动、佛系淡定、偏执疯狂、天然呆、傲娇别扭、阳光开朗、神秘莫测

**核心动机**：复仇、守护、变强、寻找真相、回到原来的世界、打破规则、证明自己的价值、活下去、还债/报恩、自由

**核心冲突**：人与体制、天才与废材、隐藏身份与暴露风险、前世与今生、规则与打破规则、科技与传统、城市与荒野、金钱与良知、虚伪与真诚、记忆与遗忘

**世界观场景**：灵气复苏的现代都市、修真宗门、末日废土、虚拟游戏世界、星际殖民地、蒸汽朋克城市、深海基地、浮空岛群、地心世界、时间循环小镇

**情节反转**：最好的朋友是最终反派、金手指其实是监控器、主角一直在反派的计划中、死去的亲人从未离开、这个世界是假的、系统背后有人在操控、反派的理由比主角更正义、主角的记忆被篡改过、穿越不是偶然而是召唤、敌人是自己未来的子孙
