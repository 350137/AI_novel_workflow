# 世界观构建

> 来源：story-cog + novelist + inkos Architect
> 使用方：Advisor (Phase 2-S1段3) / Planner / Auditor

---

## 一、世界质感锚点

来源：inkos Architect。给世界一个感官标签，不是视觉描述，是**触觉标签**：湿 / 干 / 快 / 慢 / 噪 / 静。

**为什么需要它**：一句话让读者产生身体记忆。好的质感锚点贯穿全书——每次环境描写都在强化同一个感官。

**示例**："这个世界是'干的'——荒野的风是干的、石阶上的苔是干的、力量在经脉中运转如干渴的喉咙吞咽第一口凉水、掌权者的密室是干的、连丹药都是干吞的——唯一湿的东西是汗和血。"

**其他示例**：
- 湿："这个世界是'湿的'——雾从不散，墙壁渗水，衣物永远潮冷，连火系法术打出来都带着水汽"
- 噪："这个世界是'噪的'——每个角落都有声音，安静才是异常"

## 二、7 维世界观覆盖清单

来源：story-cog 世界观构建维度清单。不是每一项都必须详细展开，但必须"有意识地被覆盖"。

| 维度 | 检查内容 |
|------|---------|
| □ 地理 | 这个世界长什么样？（大陆/海洋/气候/重要地形） |
| □ 力量/魔法体系 | 谁可以使用？有什么限制？（如不适用→标注"无"） |
| □ 社会结构 | 谁统治？什么阶层？社会流动性如何？ |
| □ 关键历史事件 | 什么塑造了现在的世界？ |
| □ 主要派系/势力 | 至少 2-3 个势力及其核心目标 |
| □ 技术水平 | 什么时代？（石器/冷兵器/蒸汽/信息/星际） |
| □ 5个有趣的地点 | 列出至少 5 个可以成为重要场景的独特地点 |

## 三、世界铁律（3-5条）

每条1句，可被 Auditor 验证。不是模糊的"力量有代价"，而是具体的可验证陈述。示例格式：
1. "跨境战斗胜利需要≥2项条件支撑（环境/法宝/战术/代价/属性克制）"
2. "从当前大境界突破到下一大境界需要的不仅是力量积累——必须有一个具体的'顿悟触发'"
3. "金手指只重塑能力基础，不能凭空创造力量——修为来自修炼，不是外挂"

## 四、力量体系结构化层级表

每个境界同时定义三个维度——缺一不可。

**表格格式**：

| 境界 | 标志（可观察的能力/现象） | 代价/限制（突破需要什么） |
|------|------------------------|--------------------------|
| 1. <境界名> | <别人怎么看出他到了这个境界> | <具体的条件——不是"修炼够久"> |
| 2. <境界名> | <标志> | <代价> |
| ... | ... | ... |

**填写原则**：
- 每个境界的"标志"必须是可被 Auditor 验证的——"散发出筑基气息"不是标志，"御剑飞行/凝气成刃"才是
- 每个境界的"代价"必须是具体的——"百日筑基" > "长期修炼"
- 天花板可控——最高境界必须有明确上限和不可逾越的代价

**跨境规则**：
- 跨境战斗需要≥2项条件支撑（环境/法宝/战术/代价/属性克制）
- 同一卷内跨境跨度≤2境
- 突破必须有铺垫（≥3章积累描写/感悟/资源获取）
- 禁止战斗中临时觉醒新能力（除非此前≥5章埋了伏笔且被 Tracker 记录）
- 每次突破后必须有适应期（≥1章）

## 五、金手指约束原则

来源：novelist。金手指不是免费午餐——必须有真实的代价。

| 维度 | 说明 |
|------|------|
| 能力天花板 | 金手指能到达的绝对上限 |
| 使用代价 | 不是"用了会累"这种伪代价。是真代价：经脉撕裂级别剧痛 / 重塑期间完全失去战斗力持续数天至数周 |
| 触发条件 | 不是任何时候都能用——需要特定环境或条件 |
| 成长路径 | 从初始→中期→完全体的完整路径 |

伪代价 vs 真代价：使用金手指重塑能力→承受经脉撕裂级别的剧痛，且重塑期间完全失去战斗力 > 使用金手指后会疲惫。
