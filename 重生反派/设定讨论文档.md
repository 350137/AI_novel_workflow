## 一、项目基础设定

| 项目 | 决定 |
|------|------|
| 书名 | 重生反派（暂定） |
| 核心模式 | 穿越 + 系统金手指 + 反套路 + 后宫 + 无敌流 |
| 参考作品 | 《玄幻：我！天命大反派》（天命王侯） |
| 世界层级 | 三界：下界 → 上界 → 仙域（不再扩展） |

## 二、境界体系（14境）

```
第1境   炼气
第2境   筑基
第3境   金丹
第4境   元婴
第5境   化神
第6境   合体
第7境   大乘
第8境   天人
第9境   法相
第10境  洞虚
第11境  圣王
第12境  道君
第13境  准帝
第14境  大帝
```

### 核心规则

- 上下界共用同一套14境体系
- 下界有境界上限（**待定**）
- 达到第14境（大帝）后，飞升/进入仙域，接仙界体系

## 四、已确认的借鉴要素

| 要素 | 来源 | 说明 |
|------|------|------|
| 穿越 | 天命大反派 | 现代人穿越到玄幻世界 |
| 系统金手指 | 天命大反派 | 类似"剧本系统"，预知气运之子行动 |
| 反套路 | 天命大反派 | 猎杀气运之子、掠夺机缘 |
| 后宫 | 天命大反派 | 多女主感情线 |
| 无敌流 | 天命大反派 | 主角实力碾压，爽文节奏 |

---

## 五、核心爽点机制（参照天命大反派）

1. 系统提前获取"剧本"，预知气运之子每一步行动
2. 在气运之子即将获得机缘时截胡
3. 以碾压之势反向猎杀，而非被动挨打
4. 掠夺气运之子的人脉、法宝、道侣
