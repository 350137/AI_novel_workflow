# 情绪曲线模板

> 基于 tomato-novelist 情绪曲线设计。Planner 规划卷级情绪分布 + Writer 控制单章情绪拐点。

---

## 核心公式：每3章完成一次"压-小扬-压-爆"循环

| 章 | 情绪阶段 | 读者心理 | 写作要点 |
|----|---------|---------|---------|
| N | 压 | "这主角太惨/太憋屈了，我要看他翻身" | 建立同情，展示困境，暗示主角有独特优点 |
| N+1 | 小扬 | "果然有反转！我就知道！" | 主角获得小胜利或重要信息，不解决根本问题 |
| N+2 | 压→爆预告 | "靠！又被打压？下章必须打脸！" | 反派变本加厉，主角获得关键筹码，章末暗示反击 |

**循环应用**：每3章循环一次，爆点规模逐轮扩大。

---

## 单章内部情绪设计

2200-2800字内至少3次情绪变化：

```
开头（50-250字）：延续或急转，快速建立本章情绪基调
  例：直接承接上章钩子，或用一个反常细节打破预期

中间（1500-1900字）：情绪逐步升级，至少一次小起伏
  例：紧张→缓解→新紧张→更紧张

结尾（450-650字）：情绪到达本章峰值后急转，留下钩子
  例：暗爽→震惊→不安
```

---

## 情绪值刻度（1-10）

```
1-2 = 压抑/日常/蓄力
3-4 = 线索展开/冲突萌芽/小进展
5-6 = 小爽点/正面进展/关系突破
7-8 = 中爽点/境界突破/重大打脸
9-10 = 卷末高潮/核心伏笔回收
```

---

## 卷级情绪曲线模板

```
章号  N1  N2  N3  N4  N5  N6  N7  N8  N9  N10 ... N-1  N
情绪  3   2   2   4   5   3   2   3   4   5   ...  8    5→3
      │   │   │   │   │   │   │   │   │   │       │    │
      开篇 日常 线索 蓄压 突破 后效 低谷 蓄压 线索 中爽   高潮  后效→钩子
```

**校验**：
- [ ] 情绪曲线有起伏（连续3章情绪值不变=节奏单调）
- [ ] 卷首情绪 ≤ 4
- [ ] 卷末情绪 ≥ 6
- [ ] 卷末章尾情绪回落2-3点（高潮冷却+下卷悬念）

---

## 评论区诱导节点

在以下节点刻意留白引发读者讨论：
1. **两难选择**：主角在两个都有道理的选项中犹豫，章末不写决定
2. **信息差**：让读者比主角多知道一点信息，但不全知道
3. **误解不解释**：主角被误解时不立刻解释
4. **配角高光**：给反派或配角一个令人同情的瞬间
5. **规则争议**：主角做出道德灰色地带的选择，不评价对错
