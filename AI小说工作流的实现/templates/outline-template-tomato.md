## 2. 节奏蓝图（每3章一个情绪单元）

| 章节区间 | 情绪任务 | 必须完成的功能 | 章末落点要求 |
|---------|---------|---------------|------------|
| 1-3章 | 压-小扬-压 | 共情困境 + 打破日常 + 展示优点 | 窒息感/好奇感，暗示反击 |
| 4-6章 | 小扬-压-爆 | 被迫行动 + 展示能力 + 获得收益 + 引爆第一雷 | "更大麻烦"或"神秘助力"暗示 |
| 7-9章 | 压-小扬-压 | 明确对立方 + 魅力配角 + 主动选择 | 抛出中期谜题 |
| 10-12章 | 小扬-压-爆 | 引爆前期雷 + 当众胜利 + 代价随至 | 旧平衡彻底打破 |
| 后续... | 循环重复，爆点规模逐轮扩大 | | |

**节奏校验**：
- [ ] 连续蓄压不超过3章 → 否则节奏单调
- [ ] 每3-5章有中爽点（小目标闭环）
- [ ] 每8-12章有大爽点（境界突破/重大打脸）
- [ ] 卷末前2-3章爬升紧张度
- [ ] 卷末高潮后有后效章冷却（≥1章）

---

## 3. 情绪标签→节奏映射

### 打脸爽文
- N（压）：因隐藏身份被嘲讽
- N+1（小扬）：不经意展露实力，被误认为运气
- N+2（压→爆预告）：遭遇更强对手，章末暗示当众揭穿

### 极致虐恋
- N（压）：被虐心/忽视
- N+1（小扬）：觉醒，做出离开决定
- N+2（压→爆预告）：对方追悔莫及

### 悬疑惊悚
- N（压）：遭遇无法解释的诡异事件
- N+1（小扬）：找到一点线索或规律
- N+2（压→爆预告）：发现规律是陷阱

### 脑洞大开
- N（压）：获得金手指但面临生存危机
- N+1（小扬）：利用金手指解决小危机
- N+2（压→爆预告）：发现金手指有代价

## 7. 评论区诱导设计

在以下节点刻意留白引发读者讨论：
1. **两难选择**：主角在两个都有道理的选项中犹豫，章末不写决定 — 第__章
2. **信息差**：让读者比主角多知道一点信息 — 第__章
3. **误解不解释**：主角被误解时不立刻解释 — 第__章
4. **配角高光**：给反派/配角一个令人同情的瞬间 — 第__章
5. **规则争议**：主角做出道德灰色地带的选择 — 第__章
