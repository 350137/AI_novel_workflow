# 钩子技巧参考

> 基于 tomato-novelist 10种悬念钩子。Writer 在章末选择钩子类型时参考。

---

## 10种钩子及示例

### 1. 突然揭示
章末突然揭示一个改变一切的信息。
> 警官看着死者的手机，最后一条短信来自一个他认识的人——他自己三天前发出的号码。

**要素**：信息出乎意料 + 改变现状理解 + 留下"为什么"

### 2. 紧急危机
角色面临迫在眉睫的危险。
> 地板开始震动，灰尘从天花板簌簌落下。她抬头一看，裂缝正在迅速扩大。

**要素**：时间紧迫 + 威胁明确 + 后果严重

### 3. 未完成的动作
一个动作被中断，留下"接下来会发生什么"。
> 他举起枪，手指扣在扳机上——"别动！"身后传来一个声音。

**要素**：动作进行中被打断 + 不确定结果 + 新变量出现

### 4. 身份反转
某人被揭示为不是我们以为的那样。
> "我终于找到你了，弟弟。"那个说着完美普通话的男人摘下面具，露出了一张她父亲的脸。

**要素**：身份误解 + 关系重定义 + 动机重新解读

### 5. 两难选择
角色面临艰难选择，章末不写决定。
> 救生艇只能载两个人。她的丈夫和女儿都在水里，海浪越来越大。她伸出手——

**要素**：选项都不理想 + 必须选择 + 高风险

### 6. 神秘物品/线索
发现一个重要但意义不明的东西。
> 保险箱里只有一张照片，拍摄于昨天。照片里是熟睡中的她，从窗外角度拍摄。

**要素**：物品意义不明 + 暗示威胁 + 激发好奇

### 7. 时间限制
一个截止时间被设定，制造紧迫感。
> 定时器显示 03:00。而拆弹包里少了一根关键的红线。

**要素**：明确时限 + 资源不足 + 后果已知

### 8. 承诺/威胁
某人做出承诺或威胁，改变预期。
> "今晚午夜之前，我会让所有人知道你十年前真正做了什么。"

**要素**：明确意图 + 伤害/揭露的威胁 + 时间框架

### 9. 离奇消失
某人或某物突然消失。
> 他转身只一秒钟，再回头时，空荡荡的牢房里，那个戴着手铐的囚犯不见了。

**要素**：不可能的行为 + 缺乏解释 + 安全感丧失

### 10. 言外之意
一句话表面正常，但暗示了更深层的东西。
> "恭喜你通过面试，"面试官笑着握住她的手，"和你的姐姐一样优秀。"可她是独生女。

**要素**：表面正常 + 隐藏信息 + 需要解读

## 钩子强度梯度

| 等级 | 类型 | 读者反应 | 适用章位 |
|-----|------|---------|---------|
| 1 | 好奇悬念 | "这很有趣" | 过渡章/蓄压早中期 |
| 2 | 关切悬念 | "接下来会发生什么" | 蓄压章/后效章 |
| 3 | 迫切悬念 | "他必须马上行动" | 爆发前章 |
| 4 | 生存悬念 | "他会活下去吗" | 高潮章/卷末-1 |
| 5 | 终极悬念 | "一切到底是什么意思" | 全书结尾 |

**原则**：强度总体上升，但允许波动。后续悬念应≥前一个。

---

## 钩子禁忌

1. **虚假悬念**：制造紧张但结果是误会 → "他听到了脚步声...原来是猫"
2. **机械降神**：突然出现从未提及的解决方案
3. **过度留白**：留下太多未回答问题 → 每章至少回应一个旧悬念
4. **低风险钩子**：结尾事件不够重要 → "他不知道晚饭吃什么"

## 钩子类型轮换

不连续3章使用同类型钩子。Planner 在每卷大纲中预分配：

| 章 | 钩子类型 | 方向 |
|----|---------|------|
| N | 悬念钩子 | |
| N+1 | 威胁钩子 | |
| N+2 | 揭示钩子 | |
