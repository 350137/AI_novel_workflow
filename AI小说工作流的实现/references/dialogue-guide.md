# 对话写作指南

> 基于 tomato-novelist 对话规范 + novelist 对话排毒。Writer 在写对话段落时参考。

---

## 核心原则

1. **每句对话必须有目的**：推动情节/揭示人物/制造冲突/传达信息/表达情感
2. **对话应该简洁**：真实的人说话不写论文
3. **真实对话不完整**：充满打断、迟疑、话题转移、话没说完

---

## 对话去书面语腔

| 书面语（避免） | 口语（使用） |
|-------------|------------|
| 为何 | 为什么 |
| 已然 | 已经 |
| 可否 | 能不能/可以吗 |
| 如此 | 这么/这样 |
| 既然……那么…… | 拆分或口语化 |
| 虽然……但是…… | 拆分或口语化 |

---

## 对话标签使用

**规则**：能辨识说话人时省略标签。用动作替代"他说"。

| 过度使用 | 改进 |
|---------|------|
| "你骗了我，"他愤怒地说。 | "你骗了我。"他的声音在颤抖。 |
| "好的，"她高兴地同意道。 | 她眼睛亮了。"好的。" |

## 角色声音区分

| 维度 | 示例 |
|-----|------|
| 用词 | 正式/俚语/方言 |
| 句式 | 长句/短句/破碎句 |
| 停顿 | 流畅/迟疑/频繁打断 |
| 语气 | 温和/激烈/冷嘲热讽 |
| 习惯语 | 特定口头禅 |

---

## 潜台词技巧

| 直接（乏味） | 潜台词（有趣） |
|-------------|-------------|
| "我很生气。" | "没事。我挺好的。真的。" |
| "我喜欢你。" | "你今天看起来...不错。" |
| "我不信任你。" | "谢谢你告诉我。我会记住的。" |

**方法**：话题转移 / 反问而非回答 / 谈论其他事物 / 沉默和动作

---

## 对话场景节奏

- **争吵场景**：短句 + 打断 + 重复强调 + 情绪升级
- **告白场景**：迟疑 + 停顿 + 寻找词语
- **审讯场景**：提问控制 + 信息不对称 + 压力建立
- **调情场景**：双关语 + 试探 + 暗示

---

## 对话与叙事穿插

- 连续对话超过8轮无叙事穿插 → 插入动作/环境/微表情
- 每3-4轮对话至少有一个非对话行（动作/环境/内心反应）
- 对话越长越礼貌 → 越短越真实。高情绪场景中对话缩短

---

## 对话常见问题

1. ❌ **信息倾倒**：角色互相说已知信息
2. ❌ **所有人说话一样**：无法区分角色
3. ❌ **过度礼貌**：真实对话更粗糙
4. ❌ **无意义闲聊**：除非揭示人物/伏笔
5. ❌ **说教**：角色发表长篇哲学论述
