## 1. 本章任务（Task List）

- [ ] **建立林剑的日常身份与性格**：通过杂物房场景展示——他是天渊城外门杂役弟子，十五岁，炼气初期，被宗门师兄弟视为废物。性格核心：犯贱吐槽是表层防御，内心深处有一个模糊但不可消除的"还不够"感。本章至少3处犯贱吐槽 + 1处独处时的安静反差。
- [ ] **建立天渊城的灰色底色**：这个世界是"饿"的——灵气稀薄，低阶修士活得艰难。杂役房的灵石配额被克扣（周师兄出场），暗示宗门以境界论人的残酷。林剑对这种不公的态度不是愤怒——是麻木的吐槽，因为"习惯了"。
- [ ] **引入矿洞方向**：林剑被罚去矿洞做杂役——从周师兄口中交代"矿洞灵气异常枯竭，没人愿意去"。林剑的第一反应是犯贱吐槽，但他的灵魂在听到"矿洞"两个字时有一瞬的异样——他不理解为什么（本章不解释）。
- [ ] **埋设H013线索（矿洞杂役身份不是巧合）**：林剑的"巧合"被分配去矿洞——看似是被周师兄欺负，实则是前世后手的因果编织。本章不揭示——只在林剑听到"矿洞"时灵魂印记微微发热（极轻，他自己都没注意到，以为是饿了）。
- [ ] **埋设H001暗线（灵魂印记存在）**：林剑独处时有一刻的恍惚——"我是不是忘了什么东西？"这种感觉一闪而过，他随即吐槽自己："忘了早饭？"——但读者应该感受到一瞬的不对劲。
- [ ] **引入配角石磊**：石磊是外门最低调的弟子之一，本章末他给林剑塞了半块灵石碎片——"这个你含着——矿洞那边灵气比这还薄，撑不住的时候含嘴里。"这是林剑日常中仅存的一点暖色。

## 2. 必须兑现（Payoffs / Must-Keep）

- [ ] **林剑的"贱"必须好笑**：至少1处让读者笑出声的犯贱吐槽——不是冷笑话，是身份反差式吐槽（废材吐槽体制、吐槽自己、吐槽现实）。
- [ ] **灰色中必须有一丝暖**：石磊的半块灵石——不煽情，但要有温度。林剑收下时的反应不能是感动——是吐槽中的停顿。"（停顿了一息）...谢了。"
- [ ] **"还不够"感必须具体化**：不能写"林剑觉得自己不该只是这样"——太抽象。要有一个具体行为：比如他睡前无意识握拳的动作、或者他看着别人练剑时不由自主紧握扫帚的手指。

## 3. 本章扣留（Held-Back）

- **扣留项1**：矿洞方向的灵气异常——林剑注意到同门说矿洞灵气枯竭，但他在那个方向感受到了一丝相反的微热（灵魂印记的轻微反应）。本章不明确揭示是后手——只给一个模糊的感官细节。读者期待："矿洞里有什么？" → 原因：催化剂在第三章 → 预计第3章揭晓。
- **扣留项2**：林剑"还不够"感的来源——本章不揭示前世、轮回、后手。只给一个模糊的内心瞬间（独处时恍惚），让读者产生好奇但不给答案。原因：本章任务是建立日常，不是打破日常。→ 预计第3章始揭。
- **扣留项3**：苏念卿完全不出现，甚至不暗示——第一卷前半段她对林剑的感知仅从远处发生，本章不应该有任何她的线索。避免过早引入神秘感压过日常建立。

## 4. 日常/过渡功能（Daily/Transition Function）

- 杂物房犯贱段落 → 功能：建立林剑的语言指纹（犯贱吐槽、自嘲式叙事）和世界观渗透（灵气分配不均、宗门以境界论人）
- 被周师兄克扣灵石 → 功能：外化"废材在这个世界的待遇"，为后文"扮猪吃虎"提供对比基准线
- 石磊给灵石 → 功能：建立"在这个灰色世界里还有人互相照应"——为石磊后续角色弧线铺第一块砖
- 林剑独处恍惚 → 功能：在搞笑节奏中打入一根冷针——提醒读者"这人不是普通废材"

## 5. 三问测试（3-Question Test）

1. **如果跳过本章，读者会错过什么？** → 错过林剑性格的全部基底——他的犯贱、他的麻木、他独处时隐约的不甘心。没有这章的打底，第三章触发后手时的"反差"就没有锚点。也错过世界观"饿"感的第一次渗透。
2. **本章的爽点/看点是什么？** → 不是打脸（本章无打脸）。看点是：①林剑的犯贱吐槽是否好笑且有人格辨识度；②灰色日常的沉浸感——读者能否感受到天渊城贫瘠的灵气和林剑麻木中的隐约不甘；③石磊塞灵石的暖色瞬间是否在灰色底色上显得珍贵。
3. **本章推进了哪条主线/支线？** → 主线：林剑前世后手系统的第一条线索（矿洞方向+灵魂印记微热）——极弱推进。支线：石磊友情线建立。世界观：天渊城灵气分配体系渗透。

## 7. 本章禁止事项（Don'ts）

- **禁止**：林剑独白解释"我觉得我不该只是这样"——必须用行为/细节/反应来表达，不能用心理独白直接说
- **禁止**：矿洞方向出现任何明确的异象（如发光、声音、明确的灵气波动）——本章只给模糊的感官细节（微热/恍惚），后手触发在第3章
- **禁止**：石磊的灵石灰煽情或过于隆重——要轻、要随意、要日常。林剑的回应可以有半息停顿，但立刻切回犯贱节奏
- **禁止**：使用"废材流"套路化开场——不能在第一章就被退婚/当众羞辱。第一章的被欺负是日常性的、系统性的（灵石配额克扣），不是戏剧性的
- **禁止**：林剑主动对"还不够"的感做出任何有意识的追问——他不能对自己说"我要变强"。在这个阶段，他觉得那只是"少吃了顿早饭"
- **禁止**：出现任何前世画面（除H017的极弱梦境碎片——星光+白衣一闪，≤2句描写）
- **疲劳词限制**："不由得""心中一凛""此子""恐怖如斯""区区"
- **AI标记词限制**："深邃的眼眸""嘴角泛起一丝""仿佛在诉说""命运的齿轮"
